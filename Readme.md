# Auto_maching_learning 自动机器学习平台
自动机器学习平台：     
自定义数据分析方式，生成可视化图表    
一键读取数据、预处理数据、训练模型、评估模型。    
生成 .py、.ipynb等格式的Python源码。


### 一、配置信息
* 运行平台:windows/linux
* python version:3.6
* 前端框架:vue 2.9.6
* 后端框架:django 3.0
* 数据库:mongodb,mysql
* 依赖库；django、numpy、pandas、sklearn、matplotlib、mongoengine
